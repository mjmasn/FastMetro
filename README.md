# FastMetro (React Native 0.56.1)

## To replicate:
```bash
git clone git@github.com:mjmasn/FastMetro.git
cd FastMetro
yarn
node node_modules/react-native/local-cli/cli.js start --reset-cache
react-native run-android
# That was fast!
```

## Setup steps:

```bash
react-native init --version 0.56.1 FastMetro
```

Added rn-cli.config.js as per @rafeca's example:
```js
const Metro = require('metro');

Metro.Logger.on('log', logEntry => {
  if (
    logEntry.action_name === 'Transforming file' &&
    logEntry.action_phase === 'end'
  ) {
    console.log(logEntry.file_name, logEntry.duration_ms);
  }
});
```

Run the build:
```bash
node node_modules/react-native/local-cli/cli.js start --reset-cache
react-native run-android
```
